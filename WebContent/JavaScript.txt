- 자바스크립트 : 웹 페이지에서 특정 동작에따른 처리가 실행되도록 도와주는 언어

1) 인터프리터 언어 <-> 컴파일 언어 (ex.자바)
  : 코드를 위에서 아래로 순차적 실행. 에러가 발생하기 전까지만 동작
2) 클라이언트 스크립트 언어
  : 서버의 부하를 덜어주는 역할 (서버에서 처리할거를 클라이언트가 처리하게 만듦)
3) '객체' 기반의 언어
4) 오픈소스기반의 코드
  : 외부로 분리는 가능하나, 완벽하게 숨길수는 없다.
5) 다양한 라이브러리 제공 - Jquery, Nodejs
6) HTML5 API 기반의 언어


* 자바스크립트 특징
- 대/소문자 구분
- 실행문의 끝에는 반드시 ;을 명시
- 가능하면 코드는 한 줄에 한 문장씩 구현
- ( 괄호 ), { 중괄호 }, [ 대괄호 ] 는 항상 짝이 맞아야 함
- "큰따옴표", '작은따옴표' 동일하게 문자 데이터를 표시함
    (속성)      (값)  에 많이 사용
